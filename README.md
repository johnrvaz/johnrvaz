# PORTFÓLIO DE PROJETOS - JOHN VAZ

Este repositório foi desenvolvido para o **Desafio: Entrega Intermediária - Criação de Repositório com Versionamento**. O objetivo é consolidar meus projetos acadêmicos e pessoais em um perfil profissional estruturado, utilizando as melhores práticas de Git e GitHub.

## 📌 Sobre o Repositório
Este projeto demonstra a organização sistemática de documentos, o uso de ferramentas de controle de versão e a integração com plataformas profissionais como LinkedIn e GitHub Pages.

### Objetivos alcançados:
* Estruturação de pastas para projetos acadêmicos e pessoais.
* Documentação detalhada via Markdown.
* Versionamento do projeto (Versão 1.0).
* Publicação via GitHub Pages.
* Integração com perfil profissional.

---

## 📂 Estrutura de Pastas
A organização do repositório segue a lógica abaixo para facilitar a navegação:

| Pasta | Conteúdo |
| :--- | :--- |
| `projetos-academicos` | Trabalhos e desafios realizados durante o curso. |
| `projetos-pessoais`   | Projetos independentes e estudos de caso. |
| `documentacao`        | Slides da apresentação, manuais e arquivos de apoio. |

## 🛠️ Ferramentas Utilizadas
As ferramentas abaixo foram fundamentais para a execução deste desafio:
* **Git**: Controle de versão local.
* **GitHub**: Hospedagem do repositório e colaboração.
* **GitHub Pages**: Publicação da página web estática.
